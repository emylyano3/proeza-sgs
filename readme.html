Proeza SGS Project 2015

Configuracion de proyecto

Proeza SGS es un proyecto Java armado con Maven, desarrollado en Eclipse y versionado con git.

Modos de importacion

Checkout via maven eclipse plugin desde el repositorio

Es necesario un conector git
  1. File > Import > Checkout Maven Proyect from SCM
  2. Seleccionar SCM Url git
  3. Ingresar la url: https://github.com/emylyano3/proeza-sgs.git
  4. Finish

Si en (2) no esta disponible la opcion git, hacer click en Find more SCM connectors in the m2e Marcketplace
